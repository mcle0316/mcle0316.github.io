Salvador Dali Illustration

This is a project that was done when I was in my first year of graphic design. We had to make a portrait of a fammous person following specific guidelines. We were to use geometric shapes and patterns, with bright colours. We also had to make elements of the portrait line up with others to create a flow through the portrait.

However, we did not pick the colors, fonts, or make the logo. That was already done when we took on the project.

Pencil Crayon Drawing

This is a drawing that I did when I was in first year. It is done using pencil crayon, on a type of canson paper that has a lot of tooth. The point of this project was to pick an old toy that is shiny and draw it, making it look as real as possible.

Tomorrowland Poster

This is a poster that was done in 4 different colours. For this specific project I was experimenting using the image trace function and life paint bucket. I started by drawing this structure that was featured at a previous tomorrowland event. Then I traced the image in illustrator. This poster was originally done in black in white, a monotone color scheme, then 2 other colors schemes with a multiple colors.

Victoria Feeds Logo

This is a logo redesign that I did for a small local business in Lindsay, Ontario called Victoria Feeds. Another name that it is known by is horse and hound. This store specializes in equestrian gear and food, but they also carry supplies for household pets and other farm animals. I decided to use the horse and the hound in the logo so that it would be easily recognized with the Horse and the Hound.

E-pub Design

This project was all about learning how to create an epub by experimenting with all the different fun things you can add to it using Adobe inDesign. We also had to create a horizontal layout for it so when you turn the ipad sideways it readjusts.

27 Dresses Poster

With this poster I was experimenting with the "flat" look. I took the title of the movie and made the poster reflect the title completely, instead of giving more of a hint for what the movie is about. I made 27 dresses in illustrator and put the together in a poster. Half of the dresses that I created were inspired by the ones that were shown in the movie.

Michael's Smart Shop

This is an app design that I made for an application for an iphone called Michael's smart shop. The purpose of this is to help users find the prices of items that do not have a tag. It also allows users to add the items scanned onto a list so they can calculate the total of their purchase after taxes before they go up to the counter. This will help prevent users from spending more money than they want to.
